------------Projeto Excel - Vendas

Este projeto apresenta um dashboard de vendas desenvolvido no Excel, utilizando dados extraídos de um banco de dados SQL.
O objetivo é demonstrar habilidades em análise de dados, visualização e integração entre banco de dados.

----------Tecnologias utilizadas
  * SQL (extração e consulta de dados)
  * Excel (visualização e construção do dashboard)
  * Gráfico estático

----------Métricas apresentadas no dashboard

O dashboard apresenta diversos indicadores de performance de vendas:
 * Receita
 * Receita mensal
 * Ticket médio por mês
 * Leads
 * Quantidade de leads gerados
 * Taxa de conversão
 * Performance por região
 * Estados que mais venderam no mês
 * Marcas mais vendidas

Top 5 marcas com maior volume de vendas:
 * Fiat
 * Chevrolet
 * Volkswagen
 * Ford
 * Renault
